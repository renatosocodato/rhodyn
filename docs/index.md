# RhoDyn documentation

RhoDyn is a standalone toolkit for dynamic residence-state analysis in live-cell
signaling and perturbation data. The documentation is organized around the
release-candidate user path.

Start with the API reference for Python use, the CLI reference for command-line
workflows, the input schema guide for table preparation, and the interpretation
guide for scientific boundaries. The reproducibility card records the current
clean-room target for Phase 6 release hardening, and the final release hardening page
records the PyPI dry run, public GitHub/Zenodo release-integrity check,
documentation-link scan, and dependency-review gates.

The RhoA/microglia manuscript and Zenodo data package remain optional reference
use cases. They are not required for normal RhoDyn installation or examples.

The Stage 7 methods-program roadmap records the future independent biological
demonstration and methods-manuscript readiness path. Stage 7.1 is complete as a
method-formalization phase through the method specification, synthetic truth
cases, limitations matrix, API notes, truth fixtures, and gate report. It does
not start independent biological analysis, benchmark-harness implementation, or
manuscript drafting.
