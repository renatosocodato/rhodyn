# Stage 3D hardening and expansion routes

Stage 3D exists to close the v0.3 public evidence bank with a bounded-coupling
case that is independent of the RhoA/microglia manuscript. The current public
ERK/Akt benchmark is sufficient for that narrow role because it provides one
passing context, two non-passing contexts, and explicit sensitivity surfaces
that keep the claim scoped.

## What is hardened now

- The source is public, DOI-backed, and CC-BY-4.0.
- Raw source archives are downloaded in memory and are not retained.
- The primary derived table contains 180 paired ERK/Akt single-cell summaries.
- UK passes the declared +/-0.20 residence-fraction bounded-coupling rule.
- S1P and histamine fail the same primary rule.
- UK passes across the 0.60 to 0.85 high-state threshold grid.
- S1P and histamine fail across the same threshold grid at the primary margin.
- UK passes at margins of 0.10 and larger, so it is not dependent on only the
  widest tested margin.
- The all-ligand pooled result is explicitly secondary because it mixes
  ligand-specific directional effects.
- Within-ligand replicate sensitivity is recorded as unavailable because the
  selected public slice has one experiment label per ligand context.

## What the result supports

The benchmark supports a context-limited bounded ERK/Akt residence contrast in
one public GPCR setting. It also shows that RhoDyn can keep other contexts
non-promoted when they fail the same declared bounded-coupling decision rule.

## What the result does not support

The benchmark does not prove biochemical equivalence between ERK and Akt, does
not exclude slower or context-specific GPCR crosstalk, and does not turn pooled
ligand behavior into a universal pathway statement.

## Higher-order routes that can wait until after Stage 4 begins

1. Add an NF-kB or ERK perturbation dataset with multiple replicate contexts to
   extend bounded coupling beyond the Wan 2021 GPCR source archive.
2. Add a public reserve-like dataset where calcium, viability, mitochondrial
   state, or recovery dynamics can be scored as a buffering coordinate.
3. Add a multi-reporter live-cell dataset where residence, amplitude, and
   routed output summaries can be compared under the same perturbations.
4. Build a Nature Methods-facing cross-system benchmark only after the backend
   can reproduce the Stage 3 outputs exactly.

These routes are valuable for the later scientific-methods campaign. They are
not required before starting the Stage 4 backend because Stage 4 should first
make the current evidence-bank surfaces executable as a service.
